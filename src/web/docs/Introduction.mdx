## DESENVOLVIMENTO WEB

### Introdução

A fim de padronizar as cores, tipografias, ícones e componentes utilizados nos produtos SaaS da NG, as equipes de produto juntamente com a equipe de pesquisa e desenvolvimento desenvolveram pacotes open source e privados para atender as necessidades das aplicações.

### Pacotes Internos

Entre os projetos desenvolvidos, encontram-se:

- biblioteca de gráficos, como o **amazing-react-charts**;
- biblioteca de componentes visuais, como o **flipper-ui**;
- biblioteca de componentes com regras de negócio específicas para os produtos SaaS, como o **frontend-core**.
