## Guia de Uso

A tipografia é utilizada para além de apresentar um design claro e eficiente, também definir uma hierarquia entre grupos de conteúdos e guiar o usuário da melhor forma possível através da interface.

A família tipográfica padrão dos produtos é a Roboto, escolhida pela versatilidade e legibilidade, como também por ser uma fonte open-source, disponível no Google Fonts.

Como o tamanho de fonte depende diretamente do tamanho do dispositivo utilizado, as tipografias web e mobile são descritas abaixo:

### Tipografia Web

A fim de desenvolver uma interface harmônica, utilizamos como base a tipografia da biblioteca `@mui`.
Para utilizá-la, a mesma é importada diretamente do `flipper-ui`:

```tsx
import React from 'react'
import { Content } from './styles'
import { Typography } from 'flipper-ui'

const App = () => (
    <Content>
        <Typography
            variant="h6"
            color="primary"
            fontWeight={600} />
    </Content>
)
```

### Tipografia Mobile

Para utilizar no projeto mobile, basta importar através do `ng-kit`, escolhendo a tipografia através do `textTheme`:

```tsx
import 'package:ng_kit/ng_kit.dart';

Text(
    'text',
    style: textTheme.bodySmall,
)
```

Para modificar a cor ou outras propriedades da tipografia predefinida, basta utilizar `copyWith` para a propriedade desejada:

```tsx
import 'package:ng_kit/ng_kit.dart';

Text(
    'text',
    style: textTheme.bodySmall?.copyWith(
        color: NGColors.darkBlue,
    )
),
```
