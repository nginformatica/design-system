## Guia de Uso

Manter um espaçamento adequado é crucial para garantir que a interface seja clara, organizada e intuitiva. O uso do 8pt Grid System, com intervalos regulares de 8px (como 8px, 16px, 24px, 32px), contribui para uma hierarquia visual bem definida entre os elementos. Para ajustes mais sutis ou em componentes menores, valores intermediários como 4px e 12px também podem ser utilizados quando necessário.

### Espaçamento Web

No desenvolvimento web, organizar todos os espaçamentos — como padding, margin e gap — em um único componente nem sempre é viável, principalmente devido à variedade de contextos em que são aplicados. Por isso, é comum utilizar os espaçamentos de forma pontual, diretamente onde são necessários, sempre seguindo a lógica de múltiplos de 4px (como 4px, 8px, 12px, 16px, etc). Essa abordagem garante consistência visual ao mesmo tempo em que oferece flexibilidade para lidar com diferentes layouts e componentes.

```tsx
import styled from '@emotion/styled'

export const Container = styled.div`
    display: grid;
    gap: 16px;
    padding: 8px;
`
```

### Espaçamento Mobile

Para utilizar no projeto mobile, basta importar através do `ng-kit`. A tag usada no componente de espaçamento é a __NGEdgeInsets__,
que deve ser usada juntamente com um dos seguintes tipos: __only__, __all__, __symmetric__ ou __onlyMixed__. Juntamente com algum desses tipos, deve ser definido
o tamanho do espaçamento utilizando o __NGSize__ com algum dos seguintes tamanhos: __s0__, __s4__, __s8__, __s12__, __s16__, __s20__, __s24__, __s28__,
__s32__, __s36__, __s40__, __s44__, __s48__, __s52__, __s56__, __s60__, __s64__. Para qualquer tamanho além de 64 pixels, deve ser utilizado a opção __onlyMixed__,
onde se pode definir livremente o valor desejado para qualquer um dos lados.

```tsx

//Usando all
Padding(
    padding:
        NGEdgeInsets.all(NGSize.s8),
    child: //Widget filho
),

//Usando symmetric
Padding(
    padding:
        NGEdgeInsets.symmetric(vertical: NGSize.s40, horizontal: NGSize.s8),
    child: //Widget filho
),

//Usando only
Padding(
    padding:
        NGEdgeInsets.only(left: NGSize.s4, top: NGSize.s8),
    child: //Widget filho
),

//Usando onlyMixed
Padding(
    padding:
        NGEdgeInsets.onlyMixed(bottom: 120, top: 12),
    child: //Widget filho
),
```
